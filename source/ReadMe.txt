1.First of all Download and install Python from here:
	https://www.python.org/downloads/
2.If you don't have pip installed then install it using this command
	sudo easy_install pip
3.Goto to the Script folder and open terminal and run this command
	pip install -r requirements.txt
4.Now we need to download chromedriver for that first of all we need to check current browser version.To check this open chrome paste the following trext and press enter
	chrome://settings/help
5.After that visit this link and download chromedriver for your current version
	http://chromedriver.chromium.org/downloads
6.After downloading extract chromedriver file from the zip to the script folder

How to use:
 1.Open urls.txt and paste urls for which you want to extract resumes.Each url must on a seperate line
 2.Double Click on script.py to start extraction

Goodluck...
